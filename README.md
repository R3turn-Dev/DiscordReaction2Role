## Usage
> 사용법

- `e;r 추가 <#채널멘션> [...메시지]`
    - 멘션된 채널에 새로운 반응-역할 메시지를 작성합니다.
    - `[...메시지]` 내용은 Embed Title 로 들어갑니다.
- `e;r 삭제 <메시지id>`
    - 등록했던 반응-역할 메시지를 삭제합니다.
